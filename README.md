## Seoul Public Bike (Ddareungi) Usage Analysis

Final project for the course **Statistical Graphics with Big Data**.

## Overview

This project analyzes factors affecting Seoul public bike (Ddareungi) usage using R programming and statistical visualization techniques.

Main variables:
- Temperature
- Rainfall
- Fine dust
- Age group
- Day of week

## Key Findings

- Higher rainfall reduced bike usage.
- Moderate temperatures increased bike usage.
- Fine dust negatively affected usage.
- People in their 20s used the service the most.
